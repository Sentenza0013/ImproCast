# 🎭 ImproCast Arena

ImproCast Arena est une application web de gestion et d'affichage de matchs d'improvisation théâtrale.

Le projet permet de piloter un match depuis une interface d'administration et d'afficher en temps réel les informations sur un écran dédié.

## 🚀 Fonctionnalités

* Gestion des équipes
* Gestion des scores
* Gestion des pénalités
* Chronomètre configurable
* Affichage temps réel
* Gestion des catégories d'improvisation
* Gestion des thèmes
* Mode plein écran
* Overlays de présentation
* Intro Arena
* Compatible PC, tablette et smartphone

## 🛠️ Technologies utilisées

* HTML5
* CSS3
* JavaScript
* LocalStorage

## 🎯 Objectif

Créer une solution moderne inspirée des tableaux d'affichage sportifs et esport pour les spectacles d'improvisation théâtrale.
